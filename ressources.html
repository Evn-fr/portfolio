Projet 1 : Réseau

Premier projet de première année de BTS SIO option SISR : le contexte Egnaro. Le but était de créer un réseau pour l'entreprise fictive Egnaro. L'infrastructure comportait 2 routeurs, 1 switch administrable et 1 switch non administrable, un client léger Axel, un point d'accès WiFi et 6 machines virtuelles : 3 serveurs Web apache2, un RDP, un serveur DHCP et un client. Ci-joint un schéma de l'architecture afin de se représenter l'architecture et son câblage.

Projet 2 : Réseau

Deuxième projet de première année de BTS SIO option SISR : le contexte Geltram. Ici il s'agit d'un infrastructure bien plus complète et permettant le bon fonctionnement d'une vraie entreprise. L'infrastructure comportait plusieurs serveurs, clients et services répartis dans 3 VLANs différents. Dans l'infrastructure, il y a un serveur WEB, un serveur HoneyPot, WSUS, un serveur AD avec le rôle DNS et DHCP, le Wiki, GLPI, FOG et PfSense + Snort.

Projet 3 : Développement

Premier projet de première année de BTS SIO en développement : la création d'un site web. Dans ce contexte, il fallait créer un site web en utilisant HTML, CCS, le framework Bootstrap afin d'avoir du responsive design et un petit peu de Javascript pour changer une image quand on clique dessus. Une fois le site terminé, il devait être intégré dans le premier projet de réseau afin qu'il soit consultable depuis le site internet de l'entreprise.

Projet 4 : Développement

Deuxième projet de première année de BTS SIO en développement : l'amélioration d'un site web existant. Pour ce projet, un élève volontaire de la classe proposait un site web d'une personne qu'il connaissait afin que la classe se sépare en plusieurs groupes et améliore le site choisi. A la fin du projet, nous avons contacté la personné dont le site web a été modifié afin qu'elle voit ce que nous étions capable de faire.

Projet 5 : Personnel

Voici un projet personnel intéressant que j'ai mené de moi-même : le rôle de "Helper" sur un serveur Minecraft. J'ai été recruté pour aider les joueurs dans leurs problèmes, les accompagner dans leurs premières heures de jeu et les aider à progresser. J'ai également rejoint le serveur Discord du serveur où j'ai pu participer à la modération et ainsi aider les joueurs dans les temps où je ne pouvait pas être présent sur le serveur Minecraft. Ce que m'a apporté cette expérience :

  • Modération d'un communauté de joueurs
  • Communication au sein d'un équipe
  • Communication envers des personnes dans le besoin
  • Gestion du stress quand beaucoup de gens demandent de l'aide
  • Les gestion des conseils et des actions : savoir donner les bons conseils plutôt que de donner des objets (qui aideront certe le joueur mais il ne sauras pas forcément quoi faire avec)
